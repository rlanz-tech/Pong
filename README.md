# Pong (1.Version)(Python 3.14.2)

Ein einfaches Pong-Spiel in Python mit der Spielebibliothek "turtle"

## Starten
python main.py

## Steuerung
W / S = Spieler 1
↑ / ↓ = Spieler 2

## Features
- 2 Spieler
- Punktesystem
- Ball- & Paddle-Kollisionen
- Schwierigkeits Erhöhung
